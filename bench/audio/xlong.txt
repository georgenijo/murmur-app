Murmur is a privacy first macOS voice to text application built with Tauri, combining a Rust backend with a React frontend for a fast and lightweight experience. Local transcription runs entirely on device using either the Parakeet transducer model through sherpa onnx or the whisper backend accelerated with Metal on Apple silicon, so no audio ever leaves the machine. The recording pipeline supports both a hold down mode and a double tap mode for dictating code comments and chat messages throughout the day.
